Invokes a draw handler to correctly lay out a ninepatch UI element.

Parameters
[in]ninepatchthe texture to draw - this should be your graphics-system's renderable texture representation.
[in]draw_handlerthe handler to use for drawing textured UI rects.
[in]extentthe position and extent of the element.
[in]src_scalethe scale of the ninepatch at source in pixels - one third of the image width.
[in]dest_scalethe scale to draw the UI element at in pixels - maps to src_scale 1:1.
[in]passthrougha passthrough for the draw handler.
Returns
an error code.
