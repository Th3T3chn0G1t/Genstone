Parses UNIX-style arguments into an easy-to-interpret format for a handler. Similar to the argp parsing library.

Parameters
[in]argcthe number of arguments to parse.
[in]argvan array of arguments.
[in]handlerthe handler to be called for each parsed argument.
[in]short_args_lengththe number of short arguments.
[in]short_argsan array of short arguments.
[in]long_args_lengththe number of long arguments.
[in]long_argsan array of long arguments.
[in]passthrougha passthrough argument for the handler.
Returns
an error code.
