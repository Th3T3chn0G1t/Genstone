Detailed Description

Contains error value enumerations. These error values are reported across the library so most headers transiently include this one.

enum gen_error_t

Return values for errorable functions. The descriptions on gen_error_t enumerations are hints, they may be used in other contexts.

Note
Some errors will become GEN_UNKNOWN on platforms that do not support them or report them nontrivially.
